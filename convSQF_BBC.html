Comment utiliser cet outil ?

  • 1. Insérez un fichier briefing.sqf dans la zone de texte de gauche 'SQF' (par glisser/déposer, en cliquant sur le bouton "Parcourir" ou par copier/coller du presse papier).
  • 2. Insérez l'entête en cliquant sur le bouton "Ajouter entête" : l'entête est ajoutée en début de texte dans la zone de texte 'SQF'.
  • 3. Saisissez les informations de l'entête (titre, map, auteur, etc.).
  • 4. Cliquez sur le bouton "Convertir" : le texte au format BBCODE est copié dans le presse papier et dans la zone de texte de droite 'BBCODE'.
  • 5. Copiez le code dans un message de forum acceptant le BBCODE (ex. grecedecanards.fr ou forum CPC).
